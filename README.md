# Neuro Consult — App Patient

Application Flutter destinée aux patients du service de neurologie pédiatrique (CHRU).

## Démo en ligne

👉 [Ouvrir la démo](https://YOUR_GITHUB_USERNAME.github.io/neuro_consult_app/)

> Remplacer `YOUR_GITHUB_USERNAME` par votre nom d'utilisateur GitHub.

## Fonctionnalités

- Questionnaire patient pré-consultation
- Génération de QR code pour transmission sécurisée des données
- Interface entièrement en français

## Stack

- Flutter 3.32+ / Dart 3.11+
- `qr_flutter`, `encrypt`, `share_plus`, `intl`

## Lancer en local

```bash
flutter pub get
flutter run -d chrome
```

## Build web

```bash
flutter build web --release
```
